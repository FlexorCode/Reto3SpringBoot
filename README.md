# Reto3SpringBoot
CRUD con Spring boot
Hecho con Apache Netbeans
Base de datos : MYSQL

Pasos para correr el programa:

1- Abrir el archivo "InsertSql" que contiene el script para crear la base de datos en MYSQL
2- Con ese script se debe crear la base de datos adoptperson
3- Import el proyecto de la carpeta BB en un IDE preferido:
   Nota: El IDE debe tener los plugins de spring boot

4- Una vez importado, clean and build el proyecto para que automaticamente instale las dependencias necesarias
5- "Run file" a la clase que esta en el package ExamReto3.BB, la clase se llama Reto3ParaElExamenApplication.java
6- Una vez ya cargado , abrir el navegador preferido y navegar a la url http://localhost:8080/
